### for Back

`go run  back/main.go`

### for Front

`$ npm i firebase`

`$ npm install -g firebase-tools`

`$ firebase login`

`$ firebase init`

`$ firebase deploy`

`$ firebase use --add`

`$ firebase serve -p 8081`
